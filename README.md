# AURORAS GeoRoute Planner

Geographic route generation tools for outdoor robots developed in the **AURORAS project**.

Project website:  
https://project-auroras.github.io/auroras/

## Overview

This repository provides a lightweight tool for generating navigation routes for outdoor robotic platforms using geographic information.

Three route generation strategies are supported:

1. **Direct GPS Route**  
   Straight-line interpolation between waypoints.

2. **OSM Road-based Route**  
   Uses OpenStreetMap road networks via OSMnx.

3. **Terrain-aware Route**  
   Uses a costmap and an A* planner combining:
   - terrain types  
   - obstacles  
   - rural tracks extracted from OpenStreetMap

The resulting routes can be exported to CSV files and visualized over satellite imagery.

## Installation

Install the dependencies:

```
pip install -r requirements.txt
```

Required Python packages include:

- folium  
- geopy  
- numpy  
- pandas  
- osmnx  
- networkx  


## Usage

Run the route generator:

```
python src/auroras_route_generator.py
```

Outputs generated by the script:

```
route_direct_straight.csv
route_road_osmnx.csv
route_costmap_astar.csv
robot_route_satellite_points.html
```

## Python Environment

It is recommended to run this project inside a Python virtual environment to avoid dependency conflicts with other Python installations on your system.

You can create and activate a virtual environment as follows:

```bash
python3 -m venv venv
source venv/bin/activate
```

Then install the required dependencies:

```bash
pip install -r requirements_venv1.txt 
```

Once the environment is activated, you can run the route generator:

```bash
python src/robot_route_generation.py
```

Using a virtual environment ensures that all required libraries are installed locally for this project and does not interfere with system-wide Python packages.


## Visualization

The generated map displays:

- **Direct route** (straight line interpolation)  
- **OSM road route** (following mapped roads and tracks)  
- **Terrain-aware route** (costmap A* planner)

Each route is drawn on top of satellite imagery and exported as CSV coordinates.

## Applications

This tool can be used for:

- outdoor robotic navigation  
- agricultural robotics  
- environmental monitoring  
- mission planning for UGVs  
- terrain-aware route planning
